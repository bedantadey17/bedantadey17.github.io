# Uninstall Edge

Posted on: 01-07-2024


Hey guys! For my first blogpost, I have a BANGER for you all. I will tell you exactly, how to do the unthinkable, the daring... HOW TO UNINSTALL MICROSOFT EDGE.

Now, before you say that you have tried it and failed miserably, let me tell you, you're not at fault. Windows makes it exceptionally difficult for any sane individual to uninstall their browser, and although Edge has some pretty dang good features, most of us prefer to use other, "safer" alternatives, like Firefox, or Brave (for all you people who can't waste three clicks to install an adblocker).

Anyways, here are the steps you need to take to uninstall Edge, the Microsoft browser that not many people (like to) use.

[!] You must install a separate browser before doing this procedure.


Method is very strong. However, future updates may affect procedure.


  • First, we have to remove the shortcut from the Start Menu. This is done so that, after uninstallation, a dead start menu item does not remain. Right-click the Edge icon in the start menu, navigate to 'Open file location', delete the shortcut. The 'Microsoft Edge' title from your start menu should be gone.

  • Navigate to C:\Program Files (x86)\Microsoft and delete every folder that starts with 'Edge'. If some files can't be deleted, try your best till very few (or ideally, none) files remain.

  • Download UninstallTool from this link or download it from my file repository (useful > UninstallTool.zip). Use it to uninstall every 'Edge'-related item. Also, commit force removal.

This should have removed every single 'Edge'-related instance on your computer. This method has been tested on Windows 10 and 11, LTSC and non-LTSC installations, and they have succeeded without fail.
